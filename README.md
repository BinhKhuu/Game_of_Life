Conway's Game of life Simulator
==========
#### Requirements
#### Requirments:
  1. IE > 8 or any other web brower
  2. Node 
  3. Node package manager

#### Install
  1. Clone the repository or download
  2. npm rm -rf node_modules
  3. npm install
  4. run npm start
  5. go to local host:3000


#### /src 
  1. index.js - main component
  2. index.css
